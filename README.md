# Exercices Python sur NumPy

Ce dépôt contient les exercices d'un cours sur NumPy.

## Installation

Créer un environnement virtuel

```bash
python3 -m venv venv
```

Activer l'environnement virtuel

```bash
venv\Scripts\activate
```

Installer NumPy

```bash
pip install numpy
```
